# 🚀 AI Task Manager

A productivity web application built with Flask.

## Features

- Task creation and management
- Kanban board (Todo / Progress / Done)
- Calendar scheduling
- Countdown timers
- XP and level gamification
- Pomodoro focus timer
- Analytics dashboard
- AI motivational assistant
- 3D animated background

## Tech Stack

Python  
Flask  
SQLite  
Chart.js  
FullCalendar  
Three.js  

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run the app:

python app.py

Open in browser:

http://127.0.0.1:5000
