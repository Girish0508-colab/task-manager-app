# 🚀 AI Task Manager

An advanced productivity web application built using Flask.

## Features

- Task creation and management
- Kanban board (Todo / Progress / Done)
- Interactive calendar view
- Task countdown timers
- XP and level gamification system
- Pomodoro focus timer
- AI motivational assistant
- Confetti celebration when tasks are completed
- 3D animated background
- Analytics dashboard

## Tech Stack

- Python
- Flask
- SQLite
- JavaScript
- Chart.js
- FullCalendar
- Three.js

## Installation

1. Clone the repository

git clone https://github.com/Girish0508-colab/task-manager-app.git

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

4. Open in browser

http://127.0.0.1:5000
